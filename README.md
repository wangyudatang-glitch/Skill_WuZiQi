# 技能五子棋

传统的五子棋，就是把五个子连成一条线，好无趣好无聊。而技能五子棋，就是在传统的五子棋上加入技能好好玩。

## 运行方式

直接用浏览器打开 `index.html`。

## 规则

### 这是一个rouge游戏，一共有7关。连成五个子，自动提子并对对手造成一点伤害。可以自由调整AI难度。每一关结束后，可以选择随机升级一个技能或获得新的技能。

#### 目前包含的技能：

第一个技能**时光倒流**，取消人和AI下的最新一手棋，退回上一步，没有CD一局可以使用三次

第二个技能**飞沙走石**，向上下左右四个方向之一移动一个自己的棋子一格，CD为10手，一局可以使用五次

第三个技能**力拔山兮**，直接提走一个AI的棋子，CD为25手，一局可以使用两次

第四个技能**棋盘爆破**，消灭选定的3*3范围内的所有棋子，CD为50手，一局可以使用两次

第五个技能**吞噬星空**，消灭场上所有棋子，CD500手，一局可以使用无限次

第六个技能**两级反转**，场上的黑子变为白子，白子变为黑子，没有CD，一局可以使用一次

第七个技能**召唤外援**，激活技能后下一次放置黑子变为黑皇后，在落子的回合，可以点击黑皇后进行移动或者吃白子。黑皇后如果连成五个子，剩下四个子被提，黑皇后仍留在棋盘上，没有CD，一局可以使用一次

第八个技能**随机落子**，激活技能后，在棋盘的一个随机的空位落一个黑子，CD为5手，一局可以使用五次

第九个技能**死亡决斗**，激活技能后，清空棋盘，在本关中把自己与AI的HP变为1并禁用自己所有技能，没有CD，一局可以使用一次
